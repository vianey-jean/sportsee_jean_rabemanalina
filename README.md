# Getting repository files
### `git clone https://github.com/vianey-jean/sportsee_jean_rabemanalina.git`


## To start app, first start Back-end

### Enter in Back-end 
### `cd Back-end`

### Install nodules
### `npm install`

### To start
### `npm start`


### Finish Back-end


## To start Front-end

### Enter in Front-end 
### `cd Front-end`

### Install nodules
### `npm install`

### To start
### `npm start`
